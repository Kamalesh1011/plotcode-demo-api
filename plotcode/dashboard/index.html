System Overview
Plotcode is an AI-assisted end-to-end software delivery automation platform. It connects feature requests (submitted via Telegram) through a 14-stage SDLC pipeline, orchestrated by specialized AI agents, with human-in-the-loop checkpoints at every critical decision.
The system is backed by MongoDB Atlas (serverless), communicates via a FastAPI backend, and provides real-time visibility through this WebSocket-connected dashboard.

RBAC Guide
Access is controlled via Telegram user IDs. Roles are assigned in the RBAC page or via Telegram /setrole command.
RoleCan Do
adminEverything — full platform control
product_ownerApprove/reject requests & plans, QA validation
developerApprove plans, merge PRs
qa_engineerQA approve/fail
requesterSubmit feature requests only
